# 🐍 Python Notes

A collection of Jupyter Notebook notes covering Python fundamentals, NumPy, and Matplotlib — written while learning and practicing for **CSE220** and beyond. Great as a quick-reference or revision guide if you're learning Python from scratch!

> 📌 **Note:** The files below cover the fundamentals. This repo is a work in progress — newer notebooks with more code, exercises, and questions will keep getting added over time. Stay tuned! 🚧

## 📂 Repository Structure

| File | Topic | What's Inside |
|---|---|---|
| [`1_python_basic.ipynb`](./1_python_basic.ipynb) | Python Basics | Variables, strings, arithmetic, input, if-else, loops, lists, tuples, dictionaries, sets, functions, lambda, imports, file I/O, try-except, OOP, `__main__` block |
| [`2_numpy_basic.ipynb`](./2_numpy_basic.ipynb) | NumPy Basics | Array creation, concatenation & stacking, slicing, shape & dimensions, indexing, copy vs. view, reshaping, transpose, iteration |
| [`2_1_numpy_brocode.ipynb`](./2_1_numpy_brocode.ipynb) | NumPy Extras | Multidimensional arrays, slicing, arithmetic operations, broadcasting, aggregate functions, boolean masking, random numbers |
| [`3_matplotlib_basic.ipynb`](./3_matplotlib_basic.ipynb) | Matplotlib Basics | Line plots, grids, markers, line styles & colors, scatter plots, bar charts (vertical/horizontal), pie charts, subplots, sine/square waves, stem plots |
| [`1_demotxt.txt`](./1_demotxt.txt) | Demo File | Sample text file used in the file I/O examples |

## 📝 Topics Covered

### Python Basics
- Variables & data types
- Strings & f-strings
- Arithmetic & comparison operations
- User input handling
- Conditionals (`if` / `elif` / `else`)
- Loops (`for`, `while`) & list comprehension
- Lists, tuples, dictionaries & sets
- Slicing (`list[start:stop:step]`)
- Functions, default & variable arguments, lambda functions
- Importing libraries (`math`, `random`) & installing packages
- File I/O (reading & appending)
- Exception handling (`try` / `except`)
- Object-Oriented Programming (classes & objects)
- The `if __name__ == '__main__':` block

### NumPy
- Creating 1D, 2D & N-dimensional arrays
- `zeros`, `ones`, `arange`, `linspace`, `empty`
- Concatenation & stacking (`vstack`)
- Array slicing & indexing (including fancy indexing)
- Shape, dimensions & reshaping
- `copy()` vs `view()`
- Transpose
- Arithmetic operations & broadcasting
- Aggregate functions (`sum`, `min`, `max`, `mean`, `std`) with axis control
- Boolean masking / filtering
- Random number generation with `default_rng`
- Iterating with `nditer`

### Matplotlib
- Basic line plots
- Grids, markers, line styles & colors
- Scatter plots
- Bar charts (vertical & horizontal)
- Pie charts (with labels)
- Subplots
- Signal plotting: sine waves, square waves, stem plots

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
   ```

2. **Install the required packages**
   ```bash
   pip install jupyter numpy matplotlib
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open any `.ipynb` file and start exploring! 🎉
